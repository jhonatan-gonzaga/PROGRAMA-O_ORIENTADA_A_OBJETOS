## 📁 Estrutura do Repositório

Este repositório contém a solução de cinco exercícios práticos que exploram conceitos fundamentais de Programação Orientada a Objetos (POO) em Python.

### 1️⃣ Sistema de Mídias Educacionais
📂 `1_Sistema_de_Midias_Educacionais/`

Implementação de um sistema para gerenciar diferentes tipos de mídias educacionais (vídeos, imagens, documentos, etc.). Este exercício trabalha com:
- Herança de classes
- Polimorfismo
- Encapsulamento de dados

---

### 2️⃣ Sistema de Funcionários de uma Empresa
📂 `2_Sistema_de_Funcionários_de_uma_Empresa/`

Desenvolvimento de um sistema para gerenciar funcionários de uma empresa com diferentes tipos de cargos e responsabilidades. Conceitos aplicados:
- Hierarquia de classes
- Métodos abstratos e concretos
- Gestão de atributos e comportamentos específicos

---

### 3️⃣ Sistema de Notificações com ABC
📂 `3_Sistema_de_Notificaçõe_scom_ABC/`

Criação de um sistema de notificações utilizando Abstract Base Classes (ABC). Este exercício demonstra:
- Uso de classes abstratas para definir interfaces
- Implementação de múltiplas estratégias de notificação
- Extensibilidade do sistema

---

### 4️⃣ Sistema de Impressão com Protocol
📂 `4_Sistema_de_Impressão_com_Protocol/`

Implementação de um sistema de impressão utilizando Protocols (Structural Subtyping). Conceitos incluem:
- Uso de Protocols para definir contratos de interface
- Flexibilidade através de duck typing
- Padrões de design estruturais

---

### 5️⃣ Sistema de Armazenamento com ABC e Protocol
📂 `5_Sistema_de_Armazenamento_com_ABC_e_Protocol/`

Sistema avançado que combina Abstract Base Classes e Protocols para criar um sistema de armazenamento robusto e extensível. Trabalha com:
- Combinação de ABC e Protocol
- Padrões de design mais complexos
- Gestão de estado e persistência

---

## 🛠️ Tecnologias Utilizadas

- **Python 3.x** - Linguagem de programação principal
- **POO Concepts** - Herança, Polimorfismo, Encapsulamento, Abstração
- **ABC (Abstract Base Classes)** - Para definição de interfaces abstratas
- **Protocol** - Para tipagem estrutural em Python

## 📋 Como Usar

Cada exercício está em seu próprio diretório. Para executar qualquer um deles:

```bash
# Navegue até o diretório do exercício desejado
cd 1_Sistema_de_Midias_Educacionais

# Execute o arquivo principal (geralmente main.py)

```

## 🎯 Objetivos de Aprendizado

Ao completar esta lista de exercícios, o aluno será capaz de:

✅ Aplicar conceitos de herança e polimorfismo  
✅ Utilizar classes abstratas (ABC) para definir contratos  
✅ Trabalhar com Protocols para tipagem estrutural  
✅ Implementar encapsulamento e abstração de dados  
✅ Criar sistemas orientados a objetos escaláveis e bem estruturados  
✅ Aplicar padrões de design em soluções práticas
